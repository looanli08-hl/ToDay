Daily Flow

让每一天
变成可见的故事

ToDay 会把 Apple Watch 与 iPhone 上分散的心率、步数、睡眠、运动与手动记录， 拼成一条能回看的生活时间轴。你不用把一天写成流水账，只需要抬手生活，剩下的交给画卷自己生成。

本地存储
健康、位置、照片和标注数据默认只留在你的设备上，不上传、不追踪。
Watch First
抬腕即记
手表端支持当前状态、快捷心情、快捷标注和微型时间线，记录动作压缩到几秒内。

自动推理，不必事事手写

系统会根据当天的睡眠、运动、步行、心率和活动数据推理出事件轮廓，让你更快看懂自己的一天。

留白保留给你亲自命名

那些无法被传感器准确解释的片段会以留白出现，你可以用一句话补上“那时候我在做什么”。

照片与情绪回到时间里

当天照片、手动心情与简短笔记会贴回对应时段，让画卷不只是健康数据，而是完整生活场景。
