# ToDay — 生活之流

## 副标题
让每一天变成可见的故事

## 分类
生活 (Lifestyle)

## App 描述
ToDay 会把你一天里的睡眠、步数、心率、运动、地点、照片和 Apple Watch 片段，整理成一张可回看的生活画卷。你不需要频繁手动记日记，App 会先被动捕捉线索，再把相邻的活动推理成一段段生活故事。对于系统暂时看不懂的留白时段，你可以再用一句标注把记忆补上。手表端支持抬手看当前状态、快速记录心情与活动，晚间再回到手机查看完整长卷。所有数据默认只存储在本地，不上传、不追踪，让记录保持私密，也更适合长期留下。

## 关键词
生活记录, 健康数据, Apple Watch, 时间线, 心率, 睡眠, 运动, 日记, 可视化

## 隐私政策要点
- 所有数据默认仅存储在本地设备
- 不上传个人健康数据、照片和地点信息
- 不接入广告追踪，不进行跨应用跟踪
